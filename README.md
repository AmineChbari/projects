## Projets

### 🗳️ Vote en temps réel
**`others/vote`** — Node.js · Socket.IO · Webpack

Application de vote en temps réel avec interface admin et interface votant. Les votes sont transmis via WebSocket, les résultats s'actualisent instantanément. Design glassmorphism.

**Points clés :** WebSocket bidirectionnel, séparation admin/votant, gestion des événements `disconnect`, validation des votes côté serveur.

---

### 🏨 Réservation de motel
**`others/motel reservation`** — Angular 21 · Node.js · Express · REST API

Application full-stack de gestion de réservations hôtelières. Front-end Angular avec formulaires réactifs, routing et services HTTP. Back-end Express avec API REST, validation et sanitisation des entrées.

**Points clés :** Angular 21 (NgModule, Router, Reactive Forms), API REST CRUD, validation serveur, proxy Angular dev, design responsive.

---

### 🛍️ Boutique en ligne
**`others/boutique`** — React · Webpack · JavaScript

Interface e-commerce construite avec React et bundlée via Webpack. Gestion de l'état, composants réutilisables, navigation entre les vues.

**Points clés :** React composants, state management, Webpack 5 configuration, CSS modules.

---

### 🎮 Pandemic (Jeu de plateau)
**`games/pandemic`** — Java · OOP

Simulation du jeu de société Pandemic en Java orienté objet. Gestion des épidémies, propagation des maladies, actions des joueurs et conditions de victoire/défaite.

**Points clés :** Héritage, encapsulation, design patterns, parsing JSON, algorithmes de propagation (DFS outbreak), optimisation O(n³) → O(n) via HashMap.

---

### 🕹️ Pong Arcade
**`games/pong`** — JavaScript · Canvas 2D · Webpack

Pong 2 joueurs en JavaScript vanilla avec thème arcade pixel. Effets néon, scanlines CRT, police Press Start 2P. Premier à 7 points gagne. La balle accélère de 8% à chaque rebond.

**Points clés :** Game loop `requestAnimationFrame`, collision AABB, héritage ES6, champs privés, thème neon/CRT.

---

### 🎯 Hit the Target
**`games/hit the target`** — JavaScript · Canvas 2D · Webpack

Jeu de tir à l'arc avec cibles mobiles, oiseaux voleurs et paquets de flèches bonus. Scoring par zone (10 à 50 pts selon la précision), rendu nocturne étoilé, flash messages animés.

**Points clés :** AABB + collision euclidienne, spawn system par compteur de frames, champs privés ES2022, sky gradient + étoiles procédurales.
